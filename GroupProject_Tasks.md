## Task Checklist

### Project Setup
- [ ] Form a group of 4–5 students
- [ ] Select a real-world data mining task/problem/dataset (e.g. from the internet)
- [ ] Set up project code in **Google Colab Jupyter Notebook**

### Data & Preprocessing
- [ ] Load and read the dataset in the notebook
- [ ] Document dataset details: features, size, source/access method
- [ ] Apply appropriate **data preprocessing** steps to improve model performance

### Model Development
- [ ] Select and justify the most appropriate ML algorithm(s) for the task
- [ ] Train and evaluate the model(s)
- [ ] Document and explain results

### PPT Preparation
- [ ] Slide 1: List each member's **contribution** (required — no contribution = zero score)
- [ ] Group members introduction slide
- [ ] Project introduction: task type, application, and problem being addressed
- [ ] Dataset explanation slide (features, size, source, how to read it, data examples)
- [ ] ML model selection slide with justification (why this model?)
- [ ] Preprocessing steps slide
- [ ] Results slide (screenshots or video demo)

### Presentation
- [ ] Each member presents their own section
- [ ] Execute and demo the code live during presentation (20–25 min total)
- [ ] Be ready for discussion/Q&A on results

### Submission
- [ ] Submit PPT and code to **Blackboard by 30 April 2026**
